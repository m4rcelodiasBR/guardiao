Ajuda & Perguntas Frequentes (FAQ)

Na página de Inventário, clique no botão verde "Novo item" no canto superior direito. Um formulário aparecerá para que você preencha os dados do bem, como número patrimonial, descrição, marca, etc. Após preencher, clique em "Salvar Item".

Localize o item que deseja transferir na tabela de Inventário. Na coluna "Ações", clique no ícone de seta (). Um formulário de transferência será exibido para que você informe o destino e uma observação. Apenas itens com status "Disponível" podem ser transferidos.

A importação em massa permite cadastrar vários itens de uma só vez a partir de um arquivo XML com formato específico. No menu, acesse a página de Importação, selecione o arquivo XML e clique em "Validar Ficheiro". O sistema irá verificar cada item. Você poderá então selecionar os itens válidos e confirmar a importação para o inventário.

O sistema utiliza "exclusão lógica". Isso significa que, ao excluir um item, ele não é apagado permanentemente, mas sim marcado como "EXCLUÍDO". Ele some da lista principal de inventário, mas seu histórico de transferências é preservado. Um item excluído pode ser "reativado" através da tela de Importação, caso um item com o mesmo número patrimonial seja importado novamente.
